# bus.aolabs.io

Minimal WRTA live tracker for routes 2, 4, 3, and 31.

Fixed location pins:
- 96 William Street
- Alden Hall
- Union Station
- Chipotle
- Cold Stone
- Blackstone Theaters

Sources:
- WRTA SWIV live vehicle data.
- WRTA SWIV stop-arrival data for the closest tracked stop after a location is selected.
- WRTA route geometry through the existing Ride Guide route endpoints.
- OpenStreetMap tiles.

- `/paper.pdf`
- Source: `manuscripts/bus_nature_style/main.tex`
